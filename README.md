# Landing Page Website

![Landing Page Preview](/img/vista-previa.jpg)

🔗 **Live Demo:** [Landing Page Website](https://javiermurua.github.io/landing-page-website/)

## 📌 Descripción

Esta es una landing page moderna y minimalista, desarrollada como práctica de diseño web. Es completamente **responsive**, asegurando una experiencia óptima en diferentes dispositivos.

## 🛠️ Tecnologías Utilizadas

- **HTML5**
- **CSS3** (Flexbox & Grid)
- **JavaScript** (interactividad básica)
- **GitHub Pages** (deployment)

## ✨ Características

✅ Diseño limpio y moderno  
✅ Totalmente responsive  
✅ Código estructurado y mantenible  
✅ Implementación en GitHub Pages  


## 🚀 Instalación y Uso

Si deseas clonar el proyecto y ejecutarlo localmente:

```sh
# Clonar el repositorio
git clone https://github.com/javiermurua/landing-page-website.git

# Entrar al directorio del proyecto
cd landing-page-website

# Abrir en el navegador (puedes simplemente abrir el archivo index.html)
```
